# Anleitung: Website-Inhalte importieren

Die Navigation und der Footer sind bereits implementiert. Die Seiten-Inhalte müssen im Admin-Bereich über den Page Builder erstellt werden.

## Vorgehen

1. **Admin-Bereich öffnen**: `/admin/pages`
2. **Neue Seite erstellen**: Klicken Sie auf "Neue Seite"
3. **Seiten-Informationen eingeben**:
   - Titel: z.B. "Babyzeichensprachekurs"
   - Slug: z.B. "babyzeichensprachekurs" (wird automatisch generiert)
   - Veröffentlicht: Aktivieren, wenn die Seite fertig ist

4. **Inhalte im Page Builder erstellen**:
   - Verwenden Sie die Block-Typen: Hero, Text, Image, Video, etc.
   - Kopieren Sie die Inhalte von der Original-Website: https://www.familien-herz-zeit.de/

## Tipp

Sie können die Inhalte von der Original-Website kopieren und im Page Builder einfügen. Verwenden Sie:
- **Hero Block** für Überschriften
- **Text Block** für Fließtext
- **Image Block** für Bilder (müssen zuerst in der Media Library hochgeladen werden)
- **Spacer Block** für Abstände

## Startseite

Die Startseite sollte folgende Elemente enthalten:
- Hero-Bereich: "Herzlich Willkommen" mit Untertitel
- Aktuelles: Nächste Kurse und Termine
- Kategorien: Babykurse, Workshops, Elternberatung, Für Fachpersonal
- Standort-Hinweis: "Aktuell in Hainichen, Mittweida & Rossau"
